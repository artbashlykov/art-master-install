# ART Master Install

Мастер-плагин WordPress для установки и управления другими расширениями ART из публичных репозиториев GitHub.

- Официальный репозиторий: https://github.com/artbashlykov/art-master-install
- Материалы автора: https://forge.artbashlykov.ru

## Возможности

- Страница **Настройки → Плагины Арта** — каталог ART-расширений
- Установка и обновление в один клик из GitHub Releases (публичные репозитории)

## Каталог (v1.4.0)

- **ART Starter** — техническая настройка сайта, визитка, 404
- **ART Editor** — редактор HTML-блоков Gutenberg, canvas, лендинги
- **ART LMS** — мини-LMS с автовыдачей цифровых продуктов и приёмом платежей

## Установка

1. Скопируйте папку `art-master-install` в `wp-content/plugins/`.
2. Активируйте **ART Master Install** в админке WordPress.
3. Откройте **Настройки → Плагины Арта**.
